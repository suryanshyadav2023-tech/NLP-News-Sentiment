# Real-Time News Sentiment Analyzer 📰🤖

An NLP-powered web application that fetches live news headlines via API and performs sentiment analysis to categorize news as **Positive**, **Negative**, or **Neutral**.

## 🚀 Overview
This project leverages Natural Language Processing (NLP) to help users quickly gauge the "mood" of the current news cycle regarding any specific topic. Instead of reading hundreds of articles, the tool provides a visual summary of sentiment trends.

## ✨ Features
- **Live Data:** Fetches real-time headlines using the NewsAPI.
- **Sentiment Scoring:** Utilizes the `TextBlob` library for polarity detection.
- **Interactive UI:** Built with Streamlit for a seamless user experience.
- **Visual Analytics:** Dynamic bar charts showing the distribution of sentiment.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **NLP Library:** TextBlob (Lexicon-based sentiment analysis)
- **Framework:** Streamlit
- **API:** NewsAPI
